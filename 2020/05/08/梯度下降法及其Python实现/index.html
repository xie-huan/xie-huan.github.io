什么是梯度下降法

梯度下降法

  • 不是一个机器学习方法
  • 是一种基于搜索的最优化方法
  • 作用：最小化一个损失函数
  • 梯度上升法：最大化一个效用函数

　　导数的正负可以代表损失函数$J$增大时参数$\theta$的方向，具体来说，就是当导数小于零时（如图中的蓝色点），$J$增大的方向即是$\theta$减小的方向，同样的，当导数大于零时，$J$增大的方向就是$\theta$增大的方向。有了方向后，目标是最小化损失函数，且由于想要使用梯度下降，所以应该在导数前乘以一个负的系数，即所谓的学习率$\eta$，用以逐步逼近损失函数的极小值，下面列举出了几个关于学习率的特点。

  • $\eta$的取值影响获得最优解的速度
  • $\eta$取值不合适，可能会得不到最优解
  • $\eta$是梯度下降法的一个超参数

总结

梯度下降法

  • 批量梯度下降法(BGD)
    • 每次都要调用所有样本
    • 每次求解速度慢，但是稳定，一定朝着梯度下降最快的方向前进
  • 随机梯度下降法(SGD, Stochastic GD)
    • 每次只看一个样本
    • 不稳定
  • 小批量梯度下降法
    • 综合上述两种方法的优点

随机的思想

  • 更快的运行速度
  • 跳出局部最优解
  • 机器学习领域很多算法都要使用随机的特点：
    • 随机搜索
    • 随机森林

再次强调

  • 不是一个机器学习算法
  • 是一种基于搜索的最优化方法
  • 作用：最小化一个损失函数
  • 梯度上升法：最大化一个效用函数
