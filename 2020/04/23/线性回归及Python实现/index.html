线性回归算法介绍

什么是线性回归

　　在统计学中，线性回归(Linear Regression)是利用称为线性回归方程的最小平方函数对一个或多个自变量和因变量之间关系进行建模的一种回归分析。这种函数是一个或多个称为回归系数的模型参数的线性组合。只有一个自变量的情况称为简单回归，大于一个自变量情况的叫做多元回归。——摘自百度百科

线性回归算法的特点

基本特点：

  • 解决回归问题（回想kNN算法解决分类问题）
  • 思想简单，容易实现
  • 许多强大的非线性模型的基础
  • 结果具有很好的可解释性
  • 蕴含机器学习中许多重要的思想

与kNN的区别：

  • kNN解决分类问题，分类问题中，数据的轴都表示特征
  • 线性回归解决回归问题，回归问题中，横轴为特征，纵轴为标签

　　样本特征只有一个，称为简单线性回归

简单线性回归的目标及解决问题的一般思路

　　我们使用一个简单的例子来说明简单线性回归的目标及其简单思路。如下图的数据，横轴代表特征，纵轴代表标签。

　　我们想要将数据点尽可能用一条直线，来拟合我们的数据点。如下图所示


　　怎么拟合才算好呢？我们用所有直线预测的点与原数据的差的平方的和作为标准，即，这个标准值越小，就说明我们求得的直线拟合效果越好。其中，，这里的a和b就是我们要求得的参数。用一句话来描述，就是，找到a和b，使尽可能小。

　　上述目标函数，也被称为损失函数（loss function），要训练好一个模型，就要最小化这个模型的损失函数的值。与之对应的是效用函数（utility function）。解决很多问题的一个思路是：

  1. 通过分析问题，确定问题的损失函数或效用函数
  2. 通过最优化损失函数或效用函数，获得机器学习的模型

　　许多机器学习算法都运用此思路，如线性回归、多项式回归、逻辑回归、SVM、神经网络等。

　　一个最优决策应具有这样的性质，不管初始状态和初始决策如何，剩下的决策（整个最优决策中的一部分）对于从这一级开始的后续多级过程，仍然是一个最优决策。应用最优性原理，可以把多级决策问题的求解过程看作是一个连续递推过程，由后向前逐级推算。在求解过程中，前一级的决策和输出状态只是后一级的初始条件（初始状态），并不影响后续子过程的最优决策。——摘自百度百科
