k近邻算法介绍

什么是k近邻算法？

　　k近邻算法，即k-Nearest Neighbor，简称kNN算法，后续内容中，我们称其为kNN算法。kNN算法适合解决分类问题，其中k表示距离测试样本最近的k个数据。找到这k个样本数据中数量最多的一类，作为最后分类的结果。

kNN算法的特点

　　kNN算法是典型的非参数学习算法，即在学习过程中无任何参数。很明显，在kNN算法中，数据即模型。kNN算法天然可以解决多回归问题。

​ kNN算法思想简单，实现起来也不复杂，但通过实现具体算法代码，我们需要掌握的是常规机器学习算法的流程。

kNN算法的流程

  1. 计算测试数据到训练数据的距离
  2. 将距离进行排序，并取出前k个训练数据
  3. 将k个训练数据对应到类别，计算每个类别的个数
  4. 找到最大个数的类别，作为算法的返回值

相关概念

　　和kNN有关的一个重要概念是：超参数。所谓超参数，就是在学习算法运行之前需要决定的参数，与之对应的是模型参数，模型参数是指在算法学习中的参数，显然，kNN算法没有模型参数。因为kNN算法只需在算法运行前，给定k值和数据，就可以进行分类预测。

　　我们如何寻找合适的超参数呢？常用方法有：领域知识、经验积累、实验搜索，由于在初学阶段，不能使用前面两种方法，实验搜索便是更适合初学者的方法。

　　其实，kNN算法不止k值这一个超参数，这将在后面的实现中进一步讨论。
