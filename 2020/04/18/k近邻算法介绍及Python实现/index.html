k近邻算法介绍

什么是k近邻算法？

　　k近邻算法，即k-Nearest Neighbor，简称kNN算法，后续内容中，我们称其为kNN算法。kNN算法适合解决分类问题，其中k表示距离测试样本最近的k个样本数据。找到这k个样本数据中数量最多的一类，作为最后分类的结果。

kNN算法的特点

　　kNN算法是典型的非参数学习算法，即在学习过程中无任何参数。很明显，在kNN算法中，数据即模型。kNN算法天然可以解决多分类问题。

　　kNN算法思想简单，实现起来也不复杂，但通过实现具体算法代码，我们需要掌握的是常规机器学习算法实现及常规从原始数据到训练模型再到评价模型好坏的全流程。

kNN算法的流程

  1. 计算测试数据到训练数据的距离
  2. 将距离进行排序，并取出前k个训练数据
  3. 将k个训练数据对应到类别，计算每个类别的个数
  4. 找到最大个数的类别，作为算法的返回值

相关概念

　　和kNN有关的一个重要概念是：超参数。所谓超参数，就是在学习算法运行之前需要决定的参数，与之对应的是模型参数，模型参数是指在算法学习过程中的参数。因为kNN算法只需在算法运行前，给定k值和数据，就可以进行分类预测，所以，kNN算法没有模型参数。

　　我们如何寻找合适的超参数呢？常用方法有：领域知识、经验积累、实验搜索，由于在初学阶段，不能使用前面两种方法，实验搜索便是更适合初学者的方法。

　　其实，kNN算法不止k值这一个超参数，这将在后面的实现中进一步讨论。

更多关于kNN的超参数

有关距离的超参数

权重

　　直到现在，我们讨论的各种实现，都只讨论了kNN算法的一个超参数：k。实际上，sklearn提供给我们的接口，不止这一个，其中另一个非常重要的超参数为是否考虑样本数据与测试样本数据之间的距离的权重。

　　上图中，绿色点代表待分类数据（测试数据），蓝色点和红色点为训练数据。如果令k为3，在不考虑距离的权重的情况下，测试数据将会被判定为蓝色点所在的类别。在考虑距离权重的情况下，测试数据离红色点更近，应考虑将测试数据分类到红色点所在的类别。

　　在计算距离的时候，我们是否仔细想过距离的定义呢？在上述过程中，我们使用的距离为欧拉距离，但实际上，两个样本之间的距离计算方法却不止这一种。明可夫斯基（Minkowski）距离作为一种广泛的距离定义在sklearn中进行实现。什么是明可夫斯基距离呢？在这里我们无需知道其具体原理，下面给出公式即可：

　　公式中的p即为kNN算法的又一个超参数，显然，只有当考虑距离权重时，给出p的值才有意义。当p为1的时候，即为曼哈顿距离，p为2的时候，即为欧拉距离。

　　其实，还有更多关于距离的定义。如向量空间余弦相似度、调整余弦相似度、皮尔森相关系数、Jaccard相似系数等。

　　n_jobs是sklearn提供给我们的有关并行的超参数，它和计算机的核数相关。其中指定-1为使用所有核并行运行kNN算法。其他更多超参数详见官方文档：sklearn官方文档。

kNN中的数据处理

为什么要进行数据处理

　　真实世界的数据复杂多样。我们从真实世界采集到的数据，往往会有数据缺失，数据尺度差距过大、数据值异常等方面的问题。同时，还可能具有数据量庞大，数据维数大，特征不具有代表性等问题。所以，在对数据进行模型训练之前，往往需要先对数据进行一定的处理。

　　针对数据维数大，通常通过PCA这种特征提取方法降维，也可以通过过滤式特征选择算法进行筛选特征。这里不讲解关于降维的方法，只简单介绍数据尺度差距问题。如下表格：

肿瘤大小（cm） 发现时间（day）
样本1 1 200
样本2 5 100

　　如使用欧拉距离计算两个样本之间的距离，就会发现样本间的距离被发现时间所主导，导致距离不能代表所有相关特征之间的距离，可能会导致分类效果急剧下降！

　　解决上述尺度问题的方法是将所有数据映射到同一尺度，其中，有两种较为常用的方法：最值归一化和均值方差归一化。下面就详细说明这两种归一化方法。

数据归一化方法

最值归一化

　　最值归一化将所有数据映射到0到1之间，计算方法为(x - min) / (max - min)，其中，x表示某一特征下的单个数据样本，min表示某一特征下数据样本的最小值，max表示某一特征下数据样本的最大值。经过这样的计算，这个特征下的每个数据样本都将映射到0到1之间，然后对每个特征进行同样的处理，这样便很好的解决了尺度问题。可以很简单地用下面的一行代码实现最值归一化方法。

　　最值归一化虽然思想简单，但它只适用于有明显边界的情况，而且，它受边界值影响较大。若样本数据中，有一个特别大的异常值数据，这样会将样本数据中的大部分映射为极小的值！

均值方差归一化

　　均值方差归一化将所有数据映射到均值为0，方差为1的分布中，计算方法为（x - mean) / s，其中，x表示某一特征下的单个数据样本，mean表示某一特征下数据样本的均值，s表示某一特征下数据样本的标准差。同样地，也可以用一行代码实现均值方差归一化方法。

　　调用sklearn中的均值方差归一化很简单，但这里有一个需要我们注意的问题。我们对测试数据集如何进行均值方差归一化呢？因为我们的测试数据可能来自真实环境中，可能某一次只有一个样本待测试，如果在测试数据上运用均值方差归一化，将变得毫无意义！因为它本身只有一个数据，标准差肯定为0！

小结

　　至此，我们已经了解了很多关于kNN算法的相关内容，是时候做一下小结，来回顾我们所学内容。kNN算法应用在实际数据上的流程可概括为：

  1. 将数据分为训练数据集、测试数据集
  2. 将数据归一化
    1. 先将训练集归一化，得到Scaler对象
    2. 将测试集用训练集得到的Scalar进行归一化
  3. 将归一化的训练数据集交给训练函数，给出模型（fit过程）
  4. 将归一化的测试数据集交给模型给出预测结果（predict过程），计算出预测准确度（score）
  5. 其中根据需要使用网格化搜索寻找最好的超参数
