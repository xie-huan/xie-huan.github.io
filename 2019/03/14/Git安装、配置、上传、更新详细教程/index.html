Git使用教程

引言

这学期移动应用开发课程要求我们使用GitHub来上传作业并提交项目GitHub链接给老师，最开始学习如何使用Git的时候，出了很多错误，不知道如何解决，跌跌撞撞在网上搜索了一下各种解决方法，但是学得不够系统化，所以，写下此博客来记录一下如何逻辑清晰的完成Git的各项操作。

安装

  • 安装地址（Windows）：https://git-for-windows.github.io/
    • Linux安装命令：sudo apt-get install git
  • Mac没用过，可以自行百度。

配置

安装完成过后，以Windows为例：
在任何目录中单击鼠标右键，找到Git Bash Here并单击：

出现命令行界面，然后输入以下命令：

git config --global user.name "Your Name"
git config --global user.email "email@example.com"

其中引号里面的内容需要填写自己的信息，如我的github账号用户名为xx790008409，邮箱为790008409@qq.com，则我的命令行应该这样写：
git config —global user.name “xx790008409”
git config —global user.email “790008409@qq.com”

简单解释一下上面的命令：git init是初始化本地项目管理文件，git add是将项目中所有修改过的文件记录到项目管理文件中，git commit是提交修改（引号内内容可以自己修改），git remote add origin是将本地项目与远程相连接（链接需要修改成你自己的），类似于创建一个关系，git push就是将本地文件真正上传到GitHub上。

这样就完成了新建一个远程仓库，并将自己的本地项目上传到远程仓库中，这时，我们打开网页中的链接，就可以看到我们上传的项目文件。
