实验条件与基础知识

互斥

1.该实验用操作系统EPOS运行于单CPU的计算机上，内核可以用开关中断实现互斥。
2.中断的开关由EFLAGS中的IF位决定：
IF=1，中断打开
IF=0，中断关闭
指令
sti，IF=1
cli，IF=0

3.一般不直接用sti/cli，而是用
save_flags_cli(flags)
保存EFLAGS的值到一个变量flags中，然后IF=0
restore_flags(flags)
把变量flags的值恢复到EFLAGS中
例：uint32_t flags;
save_flags_cli(flags);
临界区
restore_flags(flags)

线程的睡眠和唤醒

睡眠

void sleep_on(struct wait_queue **head)
参数head是睡眠队列的头指针的指针

唤醒

void wake_up(struct wait_queue **head, int n)
参数n表示要唤醒的线程个数
n小于0表示唤醒该队列中的所有线程
sleep_on和wake_up必须在关中断环境中运行
用save_flags_cli/restore_flags保护
